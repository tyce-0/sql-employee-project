# SQL Employee & Project Database

## 📌 Objective

Design and analyze a company database that manages employees, departments, and projects.

## 🛠️ Tools Used

* SQL Server

## 🧱 Database Structure

* Employee Table
* Department Table
* Project Table

## 📊 Key Queries

* Employees earning more than 60,000
* Number of employees per project
* Total salary per department
* Employee roles across projects

## 📁 Files

* schema.sql
* insert_data.sql
* queries.sql

## 📈 Insights

* Identified high-paying roles
* Tracked employee distribution across projects
* Analyzed department salary expenses
